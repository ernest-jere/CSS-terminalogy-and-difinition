CSS - Terminology & Definitions

1.       The differences between external CSS, internal CSS and inline CSS:

a.       External CSS is a CSS file linked to the HTML document from outside to enhance the presentation of HTML page content;

b.      Internal CSS is a style section in the head of the HTML document to enhance the presentation of HTML page content from within.

c.       Internal CSS is the CSS declaration within the HTML element to enhance the presentation of its content.

2.       Syntax for class and ID selectors:

a.       Class selector syntax: .class { property: value;}

b.      ID selector syntax: #ID { property: value;}

3.       You applying a single rules to two different selectors by separating them with a comma: selector1, selector2 {property: value;}

4.       Given an element that has an id of title and a class of primary, You can use both attributes for a single rule by using a chain selector: #tittle.primary { property: value;}

5.       The descendant combinator select an element that is nested within the container element.

6.       Between a rule that uses one class selector and a rule that uses three type selectors, the that uses one class selector has the higher specificity?

7.       The order of box-model properties from inside to outside: content padding border- margin.

8.       The box-sizing CSS property allows us to include the padding and border in an element's total width and height.

9.       The standard box-model, the padding and border sizes are added to the size of height and width of the element, while alternative box-model, the padding and border size are included within the height and width of the element whereby taking the available part of the width and height of the elements content.

10.  To create more space between 2 elements you use margin and not padding.

11.   To create more space between the contents of an element and its border you use padding and not margin.

12.  If you wanted two elements to overlap each other you use margin padding.

13.   Block element starts on a new line and spans the width of the page while the inline element stays on the same line with the surrounding contents.

14.   The difference between inline element and inline-block element that the inline-block element allows the setting of length and width and its bottom and top margins and paddings are respected.

15.   The h1 element is the block element.

16.   The button element is inline element.

17.   The div element is a block element.

18.   The span element is inline element.

19.   The difference between flex container and flex item is that flex container is the parent container that contains the flex items while the flex item is the nested child item inside the flex container element.

20.   You create the flex container by setting the value of its display property to flex.

21.   The 3 values defined in the shorthand flex property are flex-glow, flex-shrink and flex-basis.

22.  You make flex items arrange themselves vertically instead of horizontally by setting the flex-direction property of their container to column.

23.  The difference between justify-content and align-items is that justify-content move items horizontally while align-items moves items vertically.

24.  You use flexbox to completely center a div inside a flex container by setting the values of both justify-content and align-content properties to center.

25.  The difference between justify-content: space-between and justify-content: space-around is that the space-between value displays the flex items with space between the lines while the space-around value displays the flex items with space before, between, and after the lines.
